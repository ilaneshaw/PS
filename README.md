PS and examinePS are the core modules for the piecewise smoothing method.

• PS: can be run separately. Example code to run the module can be found in the PS.Rmd file and provides 1DPS and 2DPS predictions and maps.

• examinePS: can be run alongside the PS module to provide additional model assessment. Example code to run PS and examinePS can be found in examinePS.Rmd

It will be neccesary to install the “SpaDES” package before running any modules.
